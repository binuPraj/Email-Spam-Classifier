# Email Spam Classifier — Local Web App

This repository contains a small Flask web application that classifies email text as spam or not-spam using a simple Naive Bayes model. It includes a trainer that builds a CountVectorizer + MultinomialNB model from `emails.csv` and writes the artifacts to `model/cv.pkl` and `model/clf.pkl`.

Features
- Clean, user-friendly UI (`templates/spam_checker.html`) with example dropdowns and the result shown beneath the pasted email text.
- `train_model.py` to build model artifacts from `emails.csv` (recreate pickles for your environment).
- `app.py` will auto-train on first run if model files are missing.

Prerequisites
- Python 3.10 or newer (3.12 tested)
- Git (optional)

Quickstart (PowerShell)
```powershell
cd "C:\Users\ASUS\OneDrive\Desktop\binu\SpamEmail"
# Create and activate a virtualenv (recommended)
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# Install dependencies
python -m pip install -r requirements.txt

# (Optional) Rebuild the model from the bundled dataset
python train_model.py

# Run the web app
python app.py
```

Open your browser at: http://127.0.0.1:5000

Usage
- Paste the full email (subject + body) into the textarea or select a sample from the dropdown and press `Check`.
- The result will appear directly below the textarea and the original text will remain so you can iterate.
- Use `Check & Clear` to classify and then clear the textarea.

Files
- `app.py` — Flask application (main entrypoint)
- `train_model.py` — training script to create `model/*.pkl`
- `emails.csv` — raw dataset used by the trainer
- `model/` — contains `cv.pkl` and `clf.pkl` (generated by training)
- `templates/spam_checker.html` — improved UI template
- `requirements.txt` — Python dependencies

Troubleshooting
- If `pip install` fails on building wheels, ensure you are using a recent Python and pip. Use the pre-built `requirements.txt` in this repo.
- If you see a scikit-learn InconsistentVersionWarning, re-run `python train_model.py` inside your activated venv to re-create pickles for your installed scikit-learn version.
